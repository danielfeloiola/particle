Tube Plotter

Uma simples aplicação que plota thumbnails do YouTube em um grafo extraído do video network do YouTube Data Tools e espacializado no Gephi. Permite navegar pelo grafo usando as thumbnails de cada vídeo como nó. A versão CLI também permite fazer o download das thumbnails.

A simple web application that plots YouTube thumbnails on a graph extracted from YouTube Data Tools Video Network module and spacialized on Gephi. Allows you to visualize the graph with the thumbnails for nodes. The CLI version also allows downloading the thumbnails.

TwitData

TwitData é um simples aplicativo web que geolocaliza tweets em um mapa - permitindo buscas por usuário, hashtag ou trending topic. É uma maneira rápida de descobrir sobre o que estão falando em algum lugar, ou onde estão falando de algo.

TwitData is a simple web application that geolocates tweets on a map - wether you search by user, hashtag or trending topics. It’s a simple and fast way to find out what user are talking about somewhere - or where they are talking about something.

YouTube Data

O YouTube Data é uma ferramenta web que faz simples coletas e visualizações de dados atravéas da API do YouTube, mostrando os videos mais recomendados para um tópico específico. Também possui uma função de análise de redes sem a necessidade de baixar os dados.

YouTube Data is a web application to make simple data collections as visualizations via the YouTube API, showing what videos are more recommended for a specific search query. It also has a networks analisys function that don't require downloading the data.

Google Images Scraper

Uma simples ferramenta de linha de comando que faz a raspagem de resultados do Google Imagens. Utiliza um termo a ser pesquisado e devolve um arquivo em formato CSV e uma pasta com as imagens obtidas. Utiliza Python e Beautiful Soup.

A simple command-line interface tool that scrapes the results from Google Images. Receives a search query and returns a CSV file and a folder with the downloaded images. Requires Python and Beautiful Soup.

Vaporwave is Dead!

Uma visualização de dados analisando vídeos de músicas do gênero Vaporwave no YouTube. Nos grafos é possível observar os difertentes subgêneros do Vaporwave, como o Synthwave e Chillhop, e as relações entre eles. Criada utilizando o YouTube Data Tools, Gephi e TubePlotter.

A simple data visualizaton of Vaporwave music videos posted on YouTube. On the graphs we can see the different subgenres, like Synthwave and Chillhop, and how they relate to one another. Created using YouTube Data Tools, Gephi and TubePlotter.
